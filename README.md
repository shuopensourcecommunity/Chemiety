## 技术栈

### 后端
    SpringBoot，MySQL

### 前端
    VueJS，QuasarFramework

## 目标

* 帮助展示课程相关信息
* 提供教师上传资料和讨论区，解决信息交流，发布，查询，公告的问题

## 如何拉起前端

node v8以上

然后只需
```shell
yarn install
yarn run dev
```
前端就成功拉起了，会自动打开浏览器，并启用 auto reload。

## 如何拉起后端

首先，需要安装并且启动MySQL

然后配置好数据库相关连接
