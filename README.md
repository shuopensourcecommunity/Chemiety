## 目标

* 帮助展示课程相关信息
* 提供教师上传资料和讨论区，解决信息交流，发布，查询，公告的问题

## 如何拉起前端

首先你需要 yarn，请参照[官方网站](https://yarnpkg.com/zh-Hans/docs/install)安装yarn

然后只需
```shell
yarn install
yarn run dev
```
前端就成功拉起了，会自动打开浏览器，并启用 auto reload。
